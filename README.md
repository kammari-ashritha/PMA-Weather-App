# Aurora Weather App
**Built by Kammari Ashritha | PM Accelerator AI Engineer Internship 2026**

## Full-Stack Weather Application
React frontend + Django REST backend + MongoDB database

## Features
- Real-time current weather + 5-day forecast (OpenWeatherMap API)
- GPS auto-detection of user location
- Full CRUD — save, view, edit, delete weather searches
- Export saved searches as JSON or CSV
- Responsive design — works on desktop, tablet, mobile
- Error handling for invalid cities and API failures

## Tech Stack
- **Frontend:** React 18 + Vite + Axios
- **Backend:** Django 4.2 + Django REST Framework
- **Database:** MongoDB (via PyMongo)
- **API:** OpenWeatherMap
